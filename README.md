# 📘 Descripción del Proyecto

**TailorTrack** es un sistema de gestión integral (ERP) orientado a una tienda de indumentaria masculina sport.
El sistema busca centralizar y optimizar la administración de ventas, stock, pedidos de sastrería, clientes, proveedores y finanzas en una única plataforma.

El objetivo principal es mejorar la organización del negocio, facilitar el seguimiento de pedidos y optimizar la gestión de recursos mediante una herramienta digital accesible y eficiente.

---

# 🎯 Objetivos del Sistema

* Centralizar la gestión de ventas, stock y pedidos
* Facilitar el seguimiento de pedidos de sastrería
* Mejorar la administración de clientes y proveedores
* Optimizar el control financiero del negocio
* Brindar una interfaz clara e intuitiva para el usuario
